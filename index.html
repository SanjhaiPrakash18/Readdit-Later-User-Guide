Readdit Later is a Chrome extension that helps you save, organize, and manage your Reddit posts with ease. Unlike Reddit's built-in save feature, Readdit Later provides enhanced analytics, advanced filtering, and a beautiful interface to manage your saved content.

Installation

From Chrome Web Store

  1. Visit the Chrome Web Store
  2. Search for "Readdit Later"
  3. Click "Add to Chrome"
  4. Confirm the installation when prompted

Manual Installation (Developer Mode)

  1. Download the extension files
  2. Open Chrome and go to chrome://extensions/
  3. Enable "Developer mode" in the top right
  4. Click "Load unpacked" and select the extension folder
  5. The extension will appear in your toolbar

Getting Started

First Setup

  1. Click the Extension Icon: Look for the Readdit Later icon in your Chrome toolbar
  2. Connect Your Reddit Account: Click "Connect Reddit Account" to authenticate
  3. Grant Permissions: Allow the extension to access your Reddit saved posts
  4. Initial Sync: The extension will automatically sync your existing saved posts
Authentication Process: You'll be redirected to Reddit's official OAuth page, where you'll log in with your Reddit credentials and grant the necessary permissions (read, save, history). You'll then be automatically redirected back to the extension.

Troubleshooting

Common Issues

Authentication Problems

Problem: Can't connect to Reddit account
Solutions:
  • Clear browser cookies for Reddit
  • Try incognito/private browsing mode
  • Ensure popup blockers aren't preventing OAuth
  • Check if Reddit is accessible
Problem: "Access Forbidden" error
Solution: Your Reddit app may need proper scopes. The extension requires read, save, and history permissions.

Sync Issues

Problem: Posts not syncing
Solutions:
  • Click "Sync Posts Now" in the popup
  • Check internet connection
  • Verify Reddit account is still connected
  • Try logging out and reconnecting
Problem: Some posts missing
Explanation: Reddit's API has limitations. Very old posts (1000+ items back) may not be accessible.

Performance Issues

Problem: Extension running slowly
Solutions:
  • Clear extension storage: Chrome Settings > Extensions > Readdit Later > Storage
  • Reduce number of saved posts by cleaning up old ones
  • Restart Chrome browser

Error Messages

  • "Background script timeout": Communication issue between extension components. Fix: Reload the extension or restart Chrome
  • "Token expired": Reddit authentication expired. Fix: Logout and reconnect your account
  • "No authorization code": OAuth flow interrupted. Fix: Clear browser cache and try authentication again

Privacy & Security

Data Handling

  • Local Storage Only: All your data stays on your device
  • No Server Storage: We don't store your Reddit data on external servers
  • Secure Authentication: Uses Reddit's official OAuth 2.0 system
  • Minimal Permissions: Only requests necessary Reddit API access

Reddit API Usage

  • Read-Only Access: Can only read your saved posts, not modify your account
  • Rate Limited: Follows Reddit's API guidelines to prevent abuse
  • Secure Tokens: Authentication tokens are encrypted and stored locally

What We Access

✅ We Access:
  • Your saved Reddit posts
  • Basic profile info (username only)
  • Post metadata (title, subreddit, score, etc.)
❌ We DON'T Access:
  • Your private messages
  • Your posting history
  • Other users' private information

Technical Requirements

Browser Requirements

  • Chrome: Version 88 or higher
  • Chromium-based: Edge, Brave, Opera (recent versions)
  • Extensions API: Manifest V3 support required

Reddit Account Requirements

  • Active Reddit Account: Must be logged into Reddit
  • API Access: Account must allow third-party app connections
  • Saved Posts: Works with Reddit's native save feature

Network Requirements

  • Internet Connection: Required for initial sync and periodic updates
  • Reddit API Access: Must be able to reach oauth.reddit.com and api.reddit.com
  • Port Access: Standard HTTPS (443) access required
